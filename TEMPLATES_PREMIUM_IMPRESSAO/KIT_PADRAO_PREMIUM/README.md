# Kit Padrao Premium

Este pacote separa um template recomendado por modulo, para uso rapido.

Minha curadoria final:

- `os` -> `03-automotivo`
  Motivo: conversa melhor com oficina, laudo e linguagem visual do modulo.

- `venda` -> `02-corporativo`
  Motivo: transmite mais credibilidade comercial e funciona bem para pedido e orcamento.

- `recibo` -> `01-executivo`
  Motivo: leitura forte do valor e boa assinatura institucional.

- `recibo_pagamento` -> `02-corporativo`
  Motivo: mais limpo para comprovante financeiro.

- `fechamento_caixa` -> `03-automotivo`
  Motivo: leitura rapida de operador e diferenca, com identidade forte.

- `espelho_nota` -> `02-corporativo`
  Motivo: foco tabular e conferencial, sem excesso visual.

- `dre` -> `02-corporativo`
  Motivo: cara mais executiva para relatorio gerencial.

- `relatorio_comissao` -> `01-executivo`
  Motivo: bom equilibrio entre resumo e detalhamento por colaborador.

- `relatorio_contas_pagar` -> `02-corporativo`
  Motivo: ideal para area administrativa e financeira.

- `relatorio_contas_receber` -> `02-corporativo`
  Motivo: mesmo padrao visual de governanca financeira.

- `extrato_cliente` -> `02-corporativo`
  Motivo: passa mais formalidade ao cliente.

- `extrato_fornecedor` -> `02-corporativo`
  Motivo: combina melhor com relacionamento B2B.

- `laudo_vistoria_jrxml` -> `01-executivo`
  Motivo: melhor equilibrio entre profissionalismo, fotos e assinatura.

Como usar:

1. Abra o arquivo `padrao-recomendado` da pasta do modulo.
2. Copie o conteudo.
3. Cole na Central de Layouts ou na Central de Laudos.
4. Gere o `Preview PDF`.
5. Ajuste identidade visual, termos juridicos e rodape conforme sua operacao.
